engine.utils¶

General utility functions unrelated to but used by the game engine.

engine.utils.autoargs(cls_or_fn=None)¶

Implicitly adds the parameter signature of a given classes’ __init__ to a target __init__.

The given classes’ __init__ will be called before the target __init__.

If no class is given, the first base class will be used.

engine.utils.is_function_defined(fn)¶

Return false if the function is defined as pass, otherwise return true.

Args:

fn (function): the function to test
