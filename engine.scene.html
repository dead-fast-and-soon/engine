engine.scene¶

Contains scene-related classes.

class engine.scene.Scene(game: Game)¶

Bases: object

Represents a scene containing components.

A Scene manages a list of components and is responsible for rendering them. Internally, this uses a Batch provided by Pyglet. This reduces the amount of draw calls for all components in this Scene to one.

property component_count¶

Return the total amount of components being rendered.

This method is called when this scene is loaded.

Overriding this method eliminates the need to override __init__().

This method is called on every tick.

This method will call this scene’s batch draw, as well as every component’s on_render() methods.

Args:
